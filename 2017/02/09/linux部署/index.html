linux部署

发表于 2017-02-09 | 分类于 linux部署 | 阅读次数

elasticsearch

  • 设置jdk环境变量
  • 下载官网包,解压
  • 修改config目录下 elasticsearch.yml文件network.host配置为0.0.0.0 不然本地访问不了虚拟机的es
  • bin目录下 ./elasticsearch -d 启动es
  • ip:9200 访问es

jenkins

  • 安装包

  • 修改jenkis配置文件(不配置 导致找不到java执行目录或者权限不够)

  • service jenkins start (记得重启几次)

  • 验证码

  • 选择安装推荐的插件选项

nginx

  • 下载包 下载地址

  • 安装编译工具及库文件

  • 安装nginx

jdk

  • jdk下载地址 http://www.oracle.com/technetwork/java/javase/downloads/index.html 下载解压

  • 设置环境变量 /etc/profile

  • source /etc/profile

maven

  • maven下载地址 https://maven.apache.org/download.cgi 下载 解压

  • 设置环境变量 /etc/profile

  • source /etc/profile

mysql

  • 下载rpm wget http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm
  • sudo rpm -ivh mysql-community-release-el7-5.noarch.rpm
  • sudo yum install mysql-server

  • 登录mysql 修改密码

  • 允许ip访问

    use mysql;  
update user set host = '%'  where user ='root';  
flush privileges;
