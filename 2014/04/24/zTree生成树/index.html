zTree生成树

发表于 2014-04-24 | 分类于 前端开源框架 | 阅读次数

zTree简介

zTree 是一个依靠 jQuery 实现的多功能树插件.优异的性能,灵活的配置,多种功能的组合是 zTree 最大优点.

怎样跟业务结合生成树

zTree Api介绍

  • setting配置: 配置树展现(树生成时)
  • ztree方法: 通过ztree对象提供的方法可以动态修改树
  • treeNode: 获取树节点属性

zTree实战

为了生成上图所示的组织树,主要关注三点

  • 业务数据怎样组合成ztree所识别的父子级关系 数据
  • 树展示方式 展示方式
  • 怎样获取业务数据 架构

针对数据 setting配置

data:{
      simpleData:{
                 enable: true,
                 idKey: "id",
                 pIdKey: "pId",
                 rootPId: 0
                 }
    }

rootPId:pId为0是根节点
pIdKey:父节点id
idkey:该节点id
如下:

var treeNodes=[
               {"id":1,"pId":0,"name":"test1"},
               {"id":11,"pId":1,"name":"test11"},
               {"id":12,"pId":1,"name":"test12"},
               {"id":111,"pId":11,"name":"test111"}
              ]

treeNodes对象表示 name为test1为root层节点,test11和test12为test1的子节点 , test111为test11的子节点

针对展示方式 setting配置

var setting={
            check:{
                  enable:true,
                  chkStyle:"radio",
                  radioType:"all"
                  },
            view:{
                  dbClickExpand:false,
                  showLine:true,
                  selectMulti:false
                 }
            }

check:设置zTree的节点上是否显示checkbox / radio
view:用于在节点上固定显示自定义控件 其他配置参考官方文档

针对架构

前端加载页面 触发ajax异步调用后端接口,获取json数据 初始化zTree树

$.ajax({ 
        type:"post",
        url:"/b2b/manage/menuManage/menu/shu",
        dataType:"json",
        success:function(data){

                zNodes=data;
                var t=$("#dialog p")}
                t = $.fn.zTree.init(t, setting, zNodes);
                }
       });

后端先定义对象JsonModel(id,pId,name),然后根据业务生成树关系JsonModel列表
